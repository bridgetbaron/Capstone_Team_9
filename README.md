How to run our project from vscode: 
1. Download and unzip the project folder.
2. Open the folder in vscode.
3. Install "Live Server" extension on vscode (if not already installed).
4. click on the the index.html file and select “Open with Live Server.” This will be in the bottom right corner of your vscode
-- 
- For mobile version once in vscode open the "pages" folder and then open the "mobile pages" folder.
- You can click on any HTML file within the "mobile pages" folder to access our mobile version but indexMobile.html is the home page which is the preferble way to start.
